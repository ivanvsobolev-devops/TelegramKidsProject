# Telegram Bot

Dependency-free Node.js skeleton for the MVP parent Telegram Bot service.

## Scope

This package contains only repository bootstrap code. It does not implement bot
commands, webhook handling, parent verification, approval callbacks, or Telegram
API calls yet.

## Commands

```bash
npm --prefix services/telegram_bot install
npm --prefix services/telegram_bot run format
npm --prefix services/telegram_bot run lint
npm --prefix services/telegram_bot test
npm --prefix services/telegram_bot run build
```
