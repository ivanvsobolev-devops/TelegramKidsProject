# Architecture Decision Records

All significant technical and architectural decisions must be documented as ADRs.

Naming:

ADR-001-title.md
ADR-002-title.md

Required sections:

- Status
- Context
- Decision
- Alternatives Considered
- Consequences
- Implementation Notes

No major architectural change should be implemented without an ADR.
